User Stories:
1. As a casual consumer, I want to filter movies that other people have watched by highest rating, so I can easily find recommendations for movies to watch next.
2. As a frequent movie watcher, I want to create an account, so I can log information about the movies I've watched.
3. As someone who has friends who use the app, I want to friend their accounts, so I can see their watch history and favorite genres.

Exceptions:
1. Exception: Media title does not exist. If a user tries to search for a specific media by title that does not exist in the system, the search will give back a message stating that the title cannot be found and encourage the user to search again.
2. Exception: Duplicate media titles. If a user tries to log a piece of media that shares a title with another, they will be prompted to enter the year of release to differentiate the two entities.
3. Exception: A user wants to hide spoiler content in a review. If a user includes spoilers within a review, they will have the option to add a "contains spoilers" warning, which hides the content of their review from users who have not logged the media.
